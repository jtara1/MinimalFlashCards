# MinimalFlashCards

Create digital flash cards to help you study vocab or 
concepts as you would with physical flash cards.

build/app available for free on android in the [Play Store](https://play.google.com/store/apps/details?id=com.minimalflashcards)

## Install

1. clone this repo
2. `cd MinimalFlashCards`
2. `npm i`

probably need to `$ react-native link`

## Run

3. connect android phone w/ dev mode via usb
4. `npm start`
